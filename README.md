# BOTLEAGUE

India's National Robotics Arena Platform — connecting competitors, events, and rankings across the country.

🔗 **Live Site:** [https://botleague0.netlify.app](https://botleague0.netlify.app)

---

## Tech Stack

- **React** + **TypeScript**
- **Vite**
- **Tailwind CSS v4**

## Sections

- Hero — Live event banner, CTA buttons
- Competitions & Events — Live bracket, Upcoming events, Past results
- User Journey — 4-step path to the league
- What is BotLeague? — Features + Service overview
- Categories — Mini Makers, Junior Innovators, Young Engineers, Robo Minds
- Sports — Competition disciplines grid
- Why Register? — League advantages + Leaderboard
- Join the Ecosystem — Judge / Volunteer / Community sign-up forms
- Sponsors — Partner institutions
- Footer — Quick links + Social media

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

## Build

```bash
npm run build
```
